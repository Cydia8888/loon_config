Python 字符串数字格式化与 f-string 宽度说明

用户提问 1

详细讲解一下python字符串的数字格式化的用法，举例说明

助手回答 1

Python 字符串的数字格式化详解

在 Python 中，将数字（整数、浮点数等）嵌入字符串并控制其显示格式（如小数位数、对齐、千位分隔符等）是非常常见的需求。主要有三种方法：

1. 旧式 % 格式化（类似 C 语言 printf）
2. str.format() 方法（Python 2.6+）
3. f-string（格式化字符串字面量）（Python 3.6+，推荐）

下面逐一详细讲解，并重点介绍格式说明符的用法。

---

一、旧式 % 格式化

基本语法："格式化字符串" % (值1, 值2, ...)

1. 基本占位符

占位符 说明
%d 整数（十进制）
%f 浮点数
%e 科学计数法
%g 自动选择 %f 或 %e
%s 字符串（数字会先转成字符串）
%x 十六进制整数（小写）

```python
# 整数
print("整数：%d" % 42)          # 整数：42

# 浮点数（默认6位小数）
print("浮点数：%f" % 3.14159)   # 浮点数：3.141590

# 科学计数法
print("科学：%e" % 0.000123)    # 科学：1.230000e-04
```

2. 格式说明符语法

%[flags][width][.precision]type

· width：最小字段宽度（整数）
· .precision：小数精度（对于 %f）或有效数字（对于 %g）
· flags：
  · -：左对齐
  · +：显示正负号
  · 0：用零填充（仅数字）
  · 空格：正数前加空格
  · #：显示进制前缀（如 0x）

```python
# 宽度与填充
print("%5d" % 42)      # '   42'（宽度5，右对齐）
print("%-5d" % 42)     # '42   '（左对齐）
print("%05d" % 42)     # '00042'（零填充）

# 小数精度
print("%.2f" % 3.14159)  # '3.14'（保留两位小数）
print("%.2e" % 0.000123) # '1.23e-04'（科学计数法保留两位小数）

# 显示正负号
print("%+d" % 42)   # '+42'
print("%+d" % -42)  # '-42'
print("% d" % 42)   # ' 42'（正数前加空格）

# 千位分隔符（旧式%不支持，建议用format或f-string）
```

---

二、str.format() 方法

基本语法："{} {}".format(值1, 值2) 或带索引/名称。

1. 基本用法

```python
# 按位置
print("{} + {} = {}".format(1, 2, 3))   # 1 + 2 = 3

# 按索引
print("{0} {1}".format("Hello", "World"))  # Hello World
print("{1} {0}".format("Hello", "World"))  # World Hello

# 按名称
print("{name} is {age} years old".format(name="Tom", age=20))
```

2. 格式说明符语法

{ [索引/名称] : [填充][对齐][符号][#][0][宽度][,][.精度][类型] }

· 填充：任意字符（默认为空格）
· 对齐：
  · < 左对齐
  · > 右对齐
  · ^ 居中对齐
  · = （仅数字）将填充字符放在符号和数字之间
· 符号：
  · + 显示正负号
  · - 仅负数显示负号（默认）
  · 空格：正数前加空格
· #：显示进制前缀（0b、0o、0x）
· 0：用零填充（等价于 0>）
· 宽度：最小字段宽度
· , 或 _：千位分隔符（, 用于十进制，_ 可用于十进制、十六进制等）
· .精度：浮点数小数位数或字符串截断长度
· 类型：
  · d 十进制整数
  · f 浮点数（固定小数点）
  · e 科学计数法
  · g 通用格式
  · x / X 十六进制
  · b 二进制
  · o 八进制
  · % 百分比（数值乘以100并加%）

```python
# 整数格式
print("{:5d}".format(42))          # '   42'（宽度5，右对齐）
print("{:<5d}".format(42))         # '42   '（左对齐）
print("{:^5d}".format(42))         # ' 42  '（居中）
print("{:05d}".format(42))         # '00042'（零填充）
print("{:+d}".format(42))          # '+42'
print("{: d}".format(42))          # ' 42'（空格）
print("{:,d}".format(1234567))     # '1,234,567'（千位分隔符）
print("{:_d}".format(1234567))     # '1_234_567'（Python 3.6+）

# 浮点数
print("{:.2f}".format(3.14159))    # '3.14'（两位小数）
print("{:.2e}".format(0.000123))   # '1.23e-04'（科学计数法）
print("{:.2%}".format(0.1234))     # '12.34%'（百分比）

# 宽度+精度
print("{:8.2f}".format(3.14159))   # '    3.14'（宽度8，两位小数，右对齐）

# 十六进制/二进制
print("{:x}".format(255))          # 'ff'
print("{:X}".format(255))          # 'FF'
print("{:#x}".format(255))         # '0xff'（带前缀）
print("{:b}".format(8))            # '1000'
```

3. 对齐与填充组合

```python
# 填充字符（必须是单个字符）
print("{:*<10d}".format(42))   # '42********'（左对齐，*填充）
print("{:*>10d}".format(42))   # '********42'
print("{:*^10d}".format(42))   # '****42****'

# 数字的符号与填充（= 对齐方式）
print("{:0=+10d}".format(42))  # '+000000042'（填充在符号和数字之间）
```

---

三、f-string（格式化字符串字面量）—— 推荐

Python 3.6+ 引入，语法更简洁：f"{变量:格式说明符}"。

格式说明符与 str.format() 完全一致。

```python
num = 42
pi = 3.14159
big = 1234567

# 整数
print(f"{num:5d}")        # '   42'
print(f"{num:<5d}")       # '42   '
print(f"{num:05d}")       # '00042'
print(f"{num:+d}")        # '+42'
print(f"{big:,d}")        # '1,234,567'

# 浮点数
print(f"{pi:.2f}")        # '3.14'
print(f"{pi:.2e}")        # '3.14e+00'
print(f"{0.1234:.2%}")    # '12.34%'

# 宽度+精度
print(f"{pi:8.2f}")       # '    3.14'

# 十六进制、二进制
print(f"{255:x}")         # 'ff'
print(f"{255:#x}")        # '0xff'
print(f"{8:b}")           # '1000'

# 填充对齐
print(f"{42:*<10d}")      # '42********'
print(f"{42:*>10d}")      # '********42'
print(f"{42:*^10d}")      # '****42****'

# 表达式可以直接写在花括号内
print(f"{2 * 3 + 4:05d}") # '00010'
```

注意事项

· f-string 中的表达式不能包含反斜杠（但可先计算再放进去）
· 大括号本身用 {{ 和 }} 转义

```python
# 输出花括号
print(f"{{hello}}")  # '{hello}'
```

五、常见场景举例

1. 保留两位小数并右对齐

```python
price = 12.5
print(f"{price:>8.2f}")   # '   12.50'
```

2. 显示百分比且带符号

```python
rate = 0.8765
print(f"{rate:+.1%}")     # '+87.7%'
```

3. 货币格式（千位分隔+两位小数）

```python
amount = 1234567.891
print(f"${amount:,.2f}")  # '$1,234,567.89'
```

4. 二进制对齐

```python
for i in range(1, 9):
    print(f"{i:2d} -> {i:04b}")  # 宽度4，零填充二进制
# 输出：
#  1 -> 0001
#  2 -> 0010
# ...
```

5. 科学计数法控制有效数字

```python
num = 0.000123456
print(f"{num:.2e}")       # '1.23e-04'
```

---

六、总结建议

· 新代码请使用 f-string（Python 3.6+），最简洁高效。
· 需要兼容旧版本 Python（2.7）时使用 str.format()。
· 尽量避免使用 % 格式化（除非维护老代码）。

f-string 不仅可读性好，而且支持直接写表达式，是 Python 数字格式化的最佳实践。

用户提问 2

print(f"\r{bar} {i:3}%", end='', flush=True)这串代码中{i:3}是什么意思

助手回答 2

在 print(f"\r{bar} {i:3}%", end='', flush=True) 中，{i:3} 是 f-string 的格式说明符，含义是：

· 将变量 i 格式化为至少占 3 个字符宽度的字符串。
· 默认采用右对齐，左侧用空格填充。

举例说明：

假设 i 的值变化：

i 的值 {i:3} 输出 说明
5 '  5' 宽度3，右对齐，前面补两个空格
42 ' 42' 前面补一个空格
100 '100' 正好3位，不补空格
1000 '1000' 超过3位，输出实际宽度（不截断）

在进度条场景中的作用：

常见写法是 f"\r{bar} {i:3}%"，其中 i 是当前进度百分比（通常是整数）。使用 {i:3} 可以固定百分比的显示宽度，避免数字位数变化时导致终端输出抖动（例如从 9% 变成 10% 时，字符串长度变化会引起重绘时的视觉闪烁）。

· 若不加宽度控制：9% 和 10% 长度不同，刷新时可能残留字符。
· 加上 {i:3} 后，9% 实际输出 "  9%"（三个字符的 9 加上 %），10% 输出 " 10%"，两者总长度固定，配合 \r 回车覆盖效果更平滑。

等价写法（使用 str.format()）：

```python
"{:3}".format(i)
```

或旧式 %：

```python
"%3d" % i
```

补充说明：

· 如果需要左对齐，可以用 {i:<3}。
· 如果需要零填充，可以用 {i:03d}（但此处空格填充更常见）。
· 因为后面紧跟一个 % 字符，所以实际显示效果如 " 50%"。
